# DESAFIO  - DIO - Relatório Sales Report com categorização

O Projeto consiste em:
- Acrescentar parâmetros e utilizar a categorização nos gráficos


## Base de dados

O relatório inicial foi recuperado do desfio anterior - relatorio desavio Sales Report.pbix.


## Desenvolvimento

- Inicialmente procurei paletes de cores que trouxessem um visual mais suave.
- Criei as medidas de Totais de Discounts, Gross Sales e Profit, para criação do parâmetro.
- Criei outro parâmetro com country, product e segment.
- Adicionei os parâmetros em alguns gráficos.
- Criei os gráficos das páginas 2 e 3 baseado nas aulas
- Reorganizei os gráficos e tabelas me baseando na proporção áurea.
- Atualizei os tamanhos dos textos focando no usuário.

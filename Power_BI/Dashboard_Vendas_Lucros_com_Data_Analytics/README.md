# DESAFIO  - DIO - Relatório Sales Report

O Projeto consiste em:
- Aperfeiçoar a criação de Relatório incluindo tela de fundo
- Acrescentar as páginas 2 e 3 conforme orientação do professor
- Melhorar o tamanho e o posicionamento dos itens da página
- Analizar a proporção áurea

## Base de dados

O relatório inicial foi recuperado do repositório do GITHUB - sales_report_desafio_projeto.pbix.


## Desenvolvimento

- A página incial foi criada conforme desenvolvido em aula.
- Adicionei uma tabela calendário.
- Alterei as cores de fundo, e criei no powerpoint.
- Adicionei botões para navegação entre as páginas
- Criei os gráficos das páginas 2 e 3 baseado nas aulas
- Utilizei as métricas de cálculo de TOP 3.
- Procurei o melhor posicionamento e tamanho dos itens baseado na proporção áurea.

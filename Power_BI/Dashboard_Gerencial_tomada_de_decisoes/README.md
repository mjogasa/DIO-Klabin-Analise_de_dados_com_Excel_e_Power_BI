# DESAFIO  - DIO - Relatório com Foco no Usuário

O Projeto consiste em:
- Transformar um relatório base focando no usuário
- Acrescentar botões de navegação
- Melhorar o tamanho e o posicionamento dos itens da página
- Analizar a proporção áurea
- Criar segmentação de dados, se necessário

## Base de dados

O relatório inicial foi recuperado do repositório do GITHUB - relatorio criativo_exp_usuário.pbix.


## Desenvolvimento

- A página incial foi criada conforme desenvolvido em aula.
- Adicionei uma tabela calendário.
- Atualizei os botões para visualizar os gráficos Visão por Segmento / Visão por Produto
- Adicionei a navegação por páginas
- Nas páginas Profit e Report, alterei o posicionamento e tamanho dos itens baseado na proporção áurea.
- Atualizei o tamanho dos textos dos gráficos.

Parece um processo trivial, mas deu bastante trabalho para ajustar e atualizar todas as informações do relatório.

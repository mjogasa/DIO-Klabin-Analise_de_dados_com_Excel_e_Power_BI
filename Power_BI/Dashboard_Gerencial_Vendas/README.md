# DESAFIO  - Dashboard - Relatório Gerencial de Vendas

O Projeto consiste em apresentar uma página desenvolvida na aula e adicionar uma segunda página com:
- Segmentador de Ano: utilizando o Chiclete Slicer
- Árvore hierárquica: Soma de profits, Ano e Country
- Radar Chart: Soma do Profit por Product
- Treemap: Soma de Profit por Segment
- Gráfico de cascata: Soma de Profit por Trimestre

## Base de dados

A base de dados foi coletado da amostra do próprio Power BI - usar dados de amostra  - carregar dados de exemplo. E que também está disponível abaixo, conforme informado nas instruções de entrega.

# DESAFIO  - Dashboard - Relatório Gerencial de Vendas

O Projeto consiste em apresentar duas páginas, uma desenvolvida na aula e adicionar uma segunda página com:
- Visual de pizza: lucro por segmento
- Visual mapa 1: Soma de sales e unidades vendidas por país
- Visual mapa 2: Soma de lucro (profit) por segmento

## Base de dados

A base de dados foi coletado da amostra do próprio Power BI - usar dados de amostra  - carregar dados de exemplo. E que também está disponível abaixo, conforme informado nas instruções de entrega.
